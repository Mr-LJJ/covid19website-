# 安装Install  
建议使用XAMPP   
下载整个zip解压到你的网页服务端文件夹即可(请将所有文件放在网页根目录)   
文件包含网页代码及素材文件     
数据库需要创建一个名为ncov_web的库再导入一下面的数据表   
ncov_web.sql为数据库文件自行导入
